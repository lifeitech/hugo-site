Noise Contrastive Estimation

Noise contrastive estimation (NCE) is a method to estimate high dimensional data distributions. It converts a generative learning problem to a discriminative learning one. Parameters of the distribution are learned by doing a classification between data and noise. In this post, we introduce NCE in detail, in the setting of estimating energy models. The problem that NCE [1] addresses is to estimate some data density $p_\theta(x)$.
